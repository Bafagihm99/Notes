# COMP 1238 Notes

## Week 1 – History of Computing
- One writing system can be used by multiple languages.
- Early computing devices were mechanical before becoming electronic.
- The development of binary changed how machines process information.

## Week 2 – Bits and Bytes
- A bit is the smallest unit of data (0 or 1).
- 8 bits = 1 byte.
- Binary is base 2, unlike decimal which is base 10.

## Week 3 – Hardware Basics
- The CPU executes instructions and performs calculations.
- RAM stores temporary data while the computer is running.
- Storage devices (SSD/HDD) keep data permanently.

## Week 4 – Operating Systems
- The operating system manages hardware and software resources.
- It handles memory, processes, and file systems.
- Examples include Windows, macOS, and Linux.

## Week 5 – Networking Fundamentals
- A LAN connects devices in a small area.
- An IP address identifies devices on a network.
- The Internet is a global network of interconnected networks.
